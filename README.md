# zyx1121 marketplace

Personal Claude Code plugin marketplace by [zyx1121](https://github.com/zyx1121).

## Install

In any Claude Code session:

```
/plugin marketplace add zyx1121/marketplace
/plugin install scriptorium@zyx1121
```
